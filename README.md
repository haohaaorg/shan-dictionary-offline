# Development

## Getting Start

#### Create .env (Keep this in HaoHaa Dev team because this repo is public)

#### Install dependencies

```
npm install
```

#### Develop

```
npm run dev
```

---

## Building App

#### Build script

```
npm run build
```

#### For Mac
```
npx electron-builder
```

#### For Linux

```
npx electron-builder --linux
```

#### For Window

```
npx electron-builder --win
```
